# Prompt 6 Input - Structured Iterative Test Case Generation

## Rule Points

### Rule Point 1: VaR Platform Overview (IO-INTRO-001)
- Hong Kong Securities Clearing Company Limited (HKSCC) adopts VaR Platform to determine initial margin (IM) requirement of Clearing Participants' (CPs) portfolios
- The model contains portfolio margin component for Primary Tier (Tier P) instruments, flat rate margin component for Non-constituent Tier (Tier N) instruments, corporate action position margin component and other margin add-on components
- VaR Platform is developed in accordance with regulatory requirements and international best practices (e.g., CPMI-IOSCO Principles for Financial Market Infrastructures)

### Rule Point 2: Margin Components Description (IO-INTRO-002)
- Key Components of VaR Platform:
  - Portfolio margin component for Primary Tier (Tier P) instruments
  - Flat rate margin component for Non-constituent Tier (Tier N) instruments
  - Corporate action position margin component
  - Other margin add-on components

### Rule Point 3: Regulatory Compliance (IO-INTRO-003)
- VaR Platform is developed in accordance with regulatory requirements
- Follows international best practices (CPMI-IOSCO Principles for Financial Market Infrastructures)
- Promotes transparency through daily dissemination of risk parameters

### Rule Point 4: Document Purpose (IO-INTRO-004)
- Outlines how to use Initial Margin Risk Parameter File
- Provides calculation methodology for total MTM and margin requirement
- Applies to HKSCC clearable instruments in Hong Kong market

### Rule Point 5: Risk Parameter Dissemination (IO-INTRO-005)
- To promote transparency of the model, a file containing the key risk parameters required for calculating IM (a.k.a., "Initial Margin Risk Parameter File", or "IMRPF") will be disseminated to all HKSCC's CPs on a daily basis upon the launch of VaR Platform
